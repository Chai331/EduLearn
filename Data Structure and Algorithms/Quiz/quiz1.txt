1. Data structure is a specialized format for organizing and storing data in memory that considers:
A. Only the elements stored
B. Only the memory address
C. Both the elements stored and their relationship to each other
D. The compiler used to execute the code
Answer: C

2. According to the slides, what is the correct formula representing a program?
A. Program = Algorithm + Data Structure
B. Program = Syntax + Logic
C. Program = Input + Output
D. Program = Variables + Functions
Answer: A

3. Which of the following is an example of a Primitive Data Structure?
A. Array
B. Linked List
C. Float
D. Graph
Answer: C

4. Which category of data structure can store data of more than one type and has a size that is not fixed?
A. Primitive Data Structure
B. Non-Primitive Data Structure
C. Static Data Structure
D. Built-in Data Structure
Answer: B

5. Which of the following is a Non-Linear Data Structure?
A. Array
B. Stack
C. Queue
D. Tree
Answer: D

6. Primitive data structures cannot contain which of the following values?
A. Negative numbers
B. Zero
C. NULL value
D. Floating point values
Answer: C

7. What type of data structure stores elements sequentially or linearly?
A. Non-Linear Data Structure
B. Linear Data Structure
C. Primitive Data Structure
D. Hierarchical Data Structure
Answer: B

8. Which of the following is a Linear Data Structure?
A. Tree
B. Graph
C. Stack
D. Floating point
Answer: C

9. Which of the following is an advantage of using data structures related to fast data search?
A. Reusability
B. Abstraction
C. Efficiency
D. Polymorphism
Answer: C

10. When a data structure is compiled into a library and used by different clients, this highlights which advantage?
A. Efficiency
B. Reusability
C. Abstraction
D. Encapsulation
Answer: B
